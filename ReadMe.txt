1.) Drop the folder action script "folderAction/CantoraGramPrintFolder.scpt into "Macintosh HD/Library/Scripts/Folder Action Scripts/".
2.) Right click "printFolder" and select "Folder Actions Setup".
3.) Enable "CantoraGramPrintFolder".
4.) Pair the printer (Polaroid 00 25 C8) to the computer in the "Setup Bluetooth Device" dialogue using passcode "6000".
5.) Run "MAMP".
6.) Success!  While the screen is open, and the printer is switched on, the CantoraGram will operate.  The printer will not turn off when plugged into the wall, but will switch off after 2 minutes when running on batteries.
